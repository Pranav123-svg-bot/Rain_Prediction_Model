# Weather Forecast Data Analysis

This project analyzes **weather forecast data** using Python.  
The goal is to explore, clean, and visualize the dataset to identify useful patterns and insights.

## 📂 Project Structure
- `FINALPROJECT (1).ipynb` → Jupyter Notebook containing the analysis.
- `weather_forecast_data.csv` → Dataset used for analysis.

## 🛠️ Technologies Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🚀 How to Run
1. Clone this repository or download the notebook and dataset.
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
